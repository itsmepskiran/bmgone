# BMGOne Website

A modern, professional website for BMGOne business solutions.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: 
  - Mobile-friendly navigation menu
  - Smooth scrolling between sections
  - Contact form with validation
  - QR code integration with click-to-expand functionality
  - Parallax effects and animations
- **SEO Optimized**: Semantic HTML5 structure with proper meta tags
- **Fast Loading**: Optimized CSS and JavaScript for performance

## File Structure

```
BMGOne/
├── index.html          # Main HTML file
├── styles.css          # Complete styling
├── script.js           # Interactive JavaScript features
├── logo.webp           # BMGOne company logo
└── README.md           # This file
```

## Sections

1. **Navigation**: Fixed header with smooth scroll navigation
2. **Hero Section**: Eye-catching landing area with company introduction
3. **About Section**: Company information and key features
4. **Services Section**: Showcase of business services offered
5. **Contact Section**: Contact form and company information
6. **Footer**: Company links and copyright information

## Getting Started

1. Open `index.html` in your web browser to view the website
2. The company logo `logo.webp` is displayed throughout the site
3. All assets are self-contained - no external dependencies required

## Customization

### Colors
The color scheme is based on the logo's teal and blue palette:
- Primary Teal: `#00a8cc`
- Primary Blue: `#0077b6`
- Accent Green: `#00b4d8`
- Dark Blue: `#03045e`
- Light Teal: `#90e0ef`
- Text Dark: `#023047`
- Primary Gradient: `linear-gradient(135deg, #00a8cc 0%, #0077b6 100%)`
- Secondary Gradient: `linear-gradient(135deg, #00b4d8 0%, #03045e 100%)`

### Fonts
- Primary: Inter (Google Fonts)
- Fallback: System fonts

### Responsive Breakpoints
- Desktop: > 768px
- Tablet: 481px - 768px  
- Mobile: < 480px

## Interactive Features

- **Mobile Menu**: Hamburger menu for mobile navigation
- **Smooth Scrolling**: Click navigation links for smooth page scrolling
- **Form Validation**: Contact form with client-side validation
- **Scroll Animations**: Elements animate as they come into view
- **Parallax Effects**: Hero section has subtle parallax scrolling
- **Typing Effect**: Hero title has typing animation on load
